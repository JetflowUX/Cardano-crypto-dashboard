# Cardano Crypto Dashboard - Premium UI Kit

A high-fidelity, sophisticated UI dashboard visualizer designed for the Cardano ecosystem. This project focuses on delivering a premium user experience with modern aesthetics, glassmorphic elements, and interactive financial visualizations.

> [!NOTE]
> This is a **UI/UX-focused project**. All data shown is simulated to demonstrate the interface's capabilities and interactive states.

## ‚ú® Visual Features & Functionalities

- **Premium Interface**: A sleek, high-performance dashboard with a focus on visual excellence and modern design principles.
- **Glassmorphic Components**: Elegant sidebar and navigation elements with subtle transparency and blur effects.
- **Interactive Data Visualization**: 
    - Smooth, animated charts powered by **Recharts**.
    - Real-time visual feedback for market data simulations.
- **Financial UI Components**:
    - **Smart Portfolio Summary**: Visual breakdown of balances and accounts.
    - **Custom Credit Cards**: Custom-designed card components with unique gradients and glassmorphism.
    - **Asset Watchlist**: Performance-themed tables with hover-active states.
- **Dynamic Theming**: Flawless transition between **Dark** and **Light** modes, maintaining visual consistency across the entire UI.
- **Responsive Layout**: A robust grid system that optimizes the display of complex financial data across different screen sizes.

## üèÅ Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JetflowUX/Cardano-crypto-dashboard.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Launch the development server to explore the UI:
```bash
npm run dev
```

### Production Build

Generate a production-ready build of the UI:
```bash
npm run build
```
